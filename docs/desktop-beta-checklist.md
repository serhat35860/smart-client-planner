# Desktop Beta Checklist (Windows)

Run this checklist for every `v0.1.x-beta` build.

## Functional

- Login and register flow works.
- Client create/update/delete works.
- Task create/update/complete works.
- Note create/update/reminder complete works.
- Reports page loads and exports expected output.

## Stability

- Close and reopen app keeps persisted data.
- App starts without blank screen.
- No crash in 15-minute smoke session.
- `Open Log Folder` menu action opens logs directory.

## Update

- Update check runs on startup in packaged app.
- Downloaded update prompts for restart.
- Restart applies the new version successfully.
